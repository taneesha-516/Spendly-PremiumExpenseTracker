# Spendly

Spendly is a personal finance management web application developed as a project for **Intribit Business Solutions**. The project was completed as part of an industry-based development program, and a **Project Completion Certificate** was awarded upon its successful completion.

**Live Demo:** https://spendly-t.netlify.app/

Spendly helps users manage their finances by tracking income, expenses, budgets, savings goals, and financial insights through an intuitive dashboard while securely storing data using Firebase.

---

## Features

- Secure user authentication with Firebase
- Dashboard with financial overview
- Income and expense tracking
- Budget management
- Savings goal tracking
- Transaction history
- Interactive charts and analytics
- Financial reports
- Checklist and reminders
- Responsive design for desktop and mobile
- Secure cloud-based data storage using Firebase

## Installation

1. Clone the repository.

```bash
git clone https://github.com/taneesha-516/spendly.git
```

2. Navigate to the project directory.

```bash
cd spendly
```

3. Install dependencies (if applicable).

```bash
npm install
```

4. Configure Firebase.

Replace the Firebase configuration inside `firebaseConfig.js` with your own Firebase project credentials.

5. Run the application.

Open `index.html` in your browser or use a local development server such as Live Server.

---

## Firebase Setup

This project does not include the Firebase configuration file.

1. Create a Firebase project.
2. Copy `firebaseConfig.example.js`.
3. Rename the copy to `firebaseConfig.js`.
4. Replace the placeholder values with your own Firebase configuration.
5. Enable Firebase Authentication and Cloud Firestore before running the application.
